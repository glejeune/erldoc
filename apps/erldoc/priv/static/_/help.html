To add a project, click on Add project and fill the two fields. When click on Add. The documentation will be generated. When this is done, you will be redirected to the doc.

On the home page, juste after a project name, you see the date when the documentation was generated. If you think it's too old, clic on to refresh it. You can also click on to go to the project Github page.

At this time, we only support Github. If this is not accpetable for you, you are welcome to help us ;)
